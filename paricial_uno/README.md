En este programa desarrollé una factura sencilla en C++ que solicita el nombre completo del cliente, 
el precio de un producto y la cantidad comprada. Utilicé getline() para capturar correctamente el nombre

del cliente y cin para los demás datos. El programa calcula automáticamente el subtotal, el IVA del 12% y el total a pagar. 
También implementé fixed y setprecision(2) para mostrar los resultados con dos decimales. Finalmente, el programa muestra mi nombre 
completo y mi carné como parte de la salida requerida.
